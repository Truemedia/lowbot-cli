# Lowbot CLI
CLI tool for lowbot

## Installation
```bash
  npm i -g lowbot-cli
```

## Usage
### Creating a new lowbot project (*new* command)
#### Default starter
```bash
  lowbot new <project-name>
```
#### Custom starter
```bash
  lowbot new <project-name> <starter-name>
```
### Generating files to extend functionality (*make* command)
#### Locale
```bash
  lowbot make locale
```
#### Resolver
```bash
  lowbot make resolver
```
#### Template
```bash
  lowbot make template
```
